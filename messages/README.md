# Nest CLI codes

- **nest generate module messages**( create modules named messages inside a folder messages)

- **nest generate controller messages/messages --flat** (create messages controller and even integrate it with module file, initial message `places file in the message folder and another messages call the class messages` & --flat doesnt create any new folder controller its optional if u want more organized folder structure remove --flat )
